# ADR-0015: Open-source license (provisional)

- Status: Accepted (provisional)
- Date: 2026-07-11
- Deciders: Repository bootstrap

## Context

The repository needs a license file for contribution clarity. Final open-source posture remains an open product question.

## Decision

Retain the existing MIT license in `LICENSE` as the provisional project license for Epic 0. Revisit before public launch if trademark, content, or commercial packaging requirements demand a different license or dual-licensing model.

## Alternatives considered

- Apache-2.0 — deferred; compatible alternative if patent language becomes desirable.
- Source-available / proprietary — deferred until product and legal review.

## Consequences

- Contributions are accepted under MIT terms unless maintainers publish a change.
- License choice remains listed among open questions in the product specification.
- Any license change requires an ADR update and contributor notice.
